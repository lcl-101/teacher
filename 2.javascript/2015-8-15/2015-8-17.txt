函数的参数
作用：通过动态的改变函数体内对应的变量的值，来是同一个函数体 返回不同的结果
形参： 我们在声明函数的时候 函数括号里面 定义的变量
实参： 我们在调用函数的时候 函数括号里面传递的值就叫做  实参


参数 可以为任何类型的数据
undefined null  number string boolean object  function 

参数的个数
 形参和实参 是一一对应

 实参 < 形参  不会报错 多出来的形参 自动赋值undefined

 实参 > 形参  没有报错 多出来的实参 需要通过arguments对象去访问


arguments对象
函数创建 的时候会自动创建 arguments对象
保存的是 实参

arguments对象的属性
arguments.length   获取实参的个数
arguments.callee   获取函数的引用


函数重载
同一个函数因为参数的类型或数量不同，可以对应多个函数的实现(功能)，每种实现对应一个函数体。

函数的返回值
当我们声明一个函数的时候默认是没有返回值
 1.没有给(return 值 )语句  返回结果  undefined
 2.return 没给值  作用 退出函数
 3.return 值    返回一个值

在一个函数内 可以有多个return 语句，但是我们要把它用在 判断语句里面
function aa(){
    var a=10-20;
    return a;
    //以下代码永远都不会执行
    var b=30+30;
    return b;
}

return语句的返回值
C 返回值可以是任何的数据类型
C 只能返回一个返回值。
C 如果函数没有返回值(return )，那么这个函数的值就会自动的赋值为undefined

变量的作用域

作用域:指的就是一段代码的作用范围。
全局变量： 在整个页面的任何位置都可以访问到，作用域为全局作用域
 1.在函数外 声明的变量 为全局变量
   var aa=100;
 2.没有通过var 关键字修饰
   aa=100；
   bb;//报错
局部变量： 只能在部分代码片段(函数代码片段)中访问到
 1.在函数内通过var 关键字 声明的变量 为局部变量
 2. 函数的参数 也是一个局部变量

为什么要有 全局变量 和局部变量？
 1.提高程序逻辑性
 2.安全性
 3.减少 命名冲突
